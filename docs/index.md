我们的课程简称为 ***P2S***，有两个含义：

1. ***Learn Python The Smart Way V2***，“聪明办法学 Python 第二版”的缩写。
2. ***Prepare To Be Smart***， 我们希望同学们学习这个教程后能**学习到聪明的办法，从容的迈入人工智能的后续学习**。
