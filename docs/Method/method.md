# 授课形式

!!! warning
      暂定形式，未来可能发生变动

- 直播课，预计 24 次课左右（12 次 Chapter，12 次 Talks）
    - 以第一版的内容作为文字教材，并在此基础上重新设计 Slides
    - 讲述结合代码实操，用手写画图辅助
    - 课程录像在 Bilibili
- 作业，课后布置，对于组队学习限制时间
    - 参考 CMU 15-112
    - 使用 Vijos 进行作业评测，计划筹备课内算法比赛
- 论坛，开设相关板块用于讨论
    - 进行课内相关问答 QA，公开作业答疑
    - 相当于不定时的 Office Hour
    - 学习上的疑惑与困难互帮互助
- 结课作业，引导与挑战并存
    - 阅读《Make Your Own Neural Network》
    - 依照书籍上的内容完成 MNIST 手写数字识别
